# brand_assets

Drop client-provided brand material here **before** designing. Per CLAUDE.md, this
folder is checked first and real assets always win over placeholders.

Put here if available:
- **Logos** — SVG preferred, plus PNG fallbacks (light/dark variants)
- **Color guide** — exact hex values for the brand palette
- **Style guide** — typography, spacing, do/don't
- **Photography / imagery** — hero shots, gallery photos, textures

If a logo or palette exists here, use those exact values — do not invent brand colors.
